## このページの共有について

- **このファイルの GitHub 上の URL** を LINE などで送ると、開いた人が Markdown のプレビュー（表やリンクがそのまま見える形）で確認できます。
- 例: `https://github.com/Asabis-Inc/ASA-MEDIA-PROJECT/blob/main/STATUS.md`
- より短い URL で共有したい場合は、リポジトリで **GitHub Pages** を有効化し、この内容をトップページにすると `https://asabis-inc.github.io/ASA-MEDIA-PROJECT/` のようなリンクで共有できます。詳細は `11_Asabis/docs/Task_Management_View_And_Share_Draft.md` を参照。
